# Run this commands to execute luabla in your local machine
(Python is neccesary to be installed in your computer)

1. Move to the desired directory to work (Desktop is recommended for simplicity)
2. Clone repository with: git clone https://github.com/MarcoA-Pozol/LuablaPublic.git
3. Open a terminal (cmd/console) move to "LuablaPublic": cd LuablaPublic/
4. Create venv: python -m venv venv
5. Activate venv: venv\scripts\activate
6. Install dependencies: pip install -r requirements.txt
7. Makemigrations Authentication: python manage.py makemigrations Authentication
8. Migrate Authentication: python manage.py migrate
9. Makemigrations all modules: python manage.py makemigrations
10. Migrate all remaining modules: python manage.py migrate
11. Run server on port 10200: python manage.py runserver 10200
12. Go to your browser on the browse bar type: localhost:10200
13. ENJOY!
